# Biometric KYC Verification System

A backend-focused identity verification system integrating Smile Identity APIs with secure webhook handling and verification processing.

## 🚀 Overview

This project demonstrates the architecture and backend logic behind a biometric KYC verification workflow used for identity validation and secure processing.

The system was designed to handle verification requests, process asynchronous webhook responses, validate signatures, and securely manage verification data.

## 💡 Key Features

- Biometric KYC request handling  
- Smile Identity API integration  
- Secure webhook processing  
- Webhook signature validation  
- Verification response logging  
- Backend-driven verification workflow  

## 🛠️ Tech Stack

- PHP  
- JavaScript  
- MySQL  
- REST APIs  
- Webhook Integration
